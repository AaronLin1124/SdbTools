# SdbTools 跨平台版本开发总结

## 1 项目概述

开发一款跨平台的 DBC 转 SDB 转换工具，使用 Avalonia UI 框架（.NET 9），支持 Windows、Linux、macOS 系统。

## 2 需求清单

| # | 需求          | 说明                                                                                                 |
| - | ------------- | ---------------------------------------------------------------------------------------------------- |
| 1 | 跨平台支持    | Windows/Linux/macOS 可用                                                                             |
| 2 | Avalonia UI   | 使用 .NET 9 + Avalonia 11，不使用 WinForms                                                           |
| 3 | 打开 DBC 文件 | 从文件系统选择 .dbc 文件并解析                                                                       |
| 4 | DataGrid 显示 | 显示信号列表，包含10列：报文名称、报文ID、信号名、起始位、长度、精度、偏移量、单位、字节顺序、值类型 |
| 5 | 生成 SDB 文件 | 将有效信号转换为二进制文件，含 CRC16 校验                                                            |
| 6 | 双语切换      | 支持中文/英文界面切换                                                                                |
| 7 | 版本与更新    | 显示版本号，支持检查更新功能                                                                         |
| 8 | 无效信号处理  | CAN ID = 0 或 0xC0000000 的信号标记为"无效"，不导出到 SDB                                            |

## 3 技术实现

### 3.1 项目结构

```
SdbTools/
├── App.axaml              # 应用程序样式
├── App.axaml.cs           # 应用程序入口
├── Program.cs             # 程序入口点
├── SdbTools.csproj
├── Models/
│   └── DbcSignal.cs       # 信号数据模型
├── Parsers/
│   └── DbcParser.cs       # DBC 文件解析器
├── Writers/
│   └── BinWriter.cs       # SDB 文件生成器
└── Views/
    ├── MainWindow.axaml   # 主窗口 UI
    └── MainWindow.axaml.cs # 主窗口逻辑
```

### 3.2 核心模块

#### DbcSignal.cs

- 属性：MessageName、MessageId、Name、StartBit、Length、Factor、Offset、Unit、ByteOrder、ValueType、IsInvalid
- 计算属性：MessageIdHex（十六进制显示）、MessageIdDecimal（十进制显示）

#### DbcParser.cs

- 使用正则表达式解析 DBC 文件
- 支持十进制和十六进制 CAN ID（如 `3221225472` 或 `0xC0000000`）
- 提取信号名称、起始位、长度、精度、偏移量、单位、字节顺序、值类型

#### BinWriter.cs

- 遍历所有有效信号（过滤 CAN ID = 0 和 0xC0000000）
- 将信号数据写入二进制文件
- 末尾添加 CRC16 校验和

#### MainWindow.axaml.cs

- 语言字典：存储中英文对照文本
- DataGrid 动态列：程序化创建10列，支持语言切换
- 事件处理：打开文件、生成文件、切换语言、检查更新

## 5 Bug 修复记录

| # | Bug 描述               | 原因                                 | 解决方案                                                    |

| 1 | 十进制 CAN ID 解析失败 | 正则表达式只匹配十六进制 `0x` 开头 | 修改正则，支持匹配十进制数字如 `3221225472`               |
| 2 | "检查升级"按钮未翻译   | 翻译数组缺少该词                     | 添加到语言字典，索引位置 [25]                               |
| 3 | "Check Update"显示不全 | 按钮宽度 80px 不够                   | 将按钮宽度增加到 130px                                      |
| 4 | 按钮文字不居中         | 缺少对齐属性                         | 添加 HorizontalContentAlignment 和 VerticalContentAlignment |
